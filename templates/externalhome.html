About LiteWatch Web Crawler

LiteWatch is a web-crawler program where it surfs through the valid websites and notifies the user of keyword matches it registers.

Principles of Litewatch:

  • Look Up - User Inputs Keyword
  • Investigate - Searches listings matching keyword
  • Transcribe - Record matching hits
  • Enunciate - Report Findings to user
